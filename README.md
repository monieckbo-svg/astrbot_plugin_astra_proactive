# Astra主动消息 v1.4.1

模型通过工具自主安排联系。不是系统催模型发消息，是模型自己调用工具说"30分钟后去找她"。

## 工作原理

1. 正常聊天时，模型觉得需要之后找你 → 调用 `plan_next_message` 工具
2. 工具记录计划，模型看到返回值"记住了，30分钟后去找宝宝"
3. 30分钟后，决策层（便宜模型）判断现在适不适合发
4. 适合 → 主模型生成消息 → 发送 → 可以再安排下一次 → 循环

模型不调用工具，插件就安静待着，什么都不干。

## 安装

1. 文件夹放到 `data/plugins/`
2. WebUI里配置 target_qq 和决策层provider
3. 确保AstrBot的 **function tools 开关是开着的**
4. 重启

不需要改system prompt，工具描述本身会告诉模型怎么用。

## 调试

- `/astra_status` 查看当前状态和计划
- `/astra_now` 手动触发一次
